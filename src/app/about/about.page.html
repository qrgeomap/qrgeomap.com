QR geomap is a project that aims to provide the easiest way to use and share basic maps for outdoor activities.

The idea is to use a single image file, which will logically contain the map of an area (possibly with a track, markers or any other annotations directly painted on the image), but the image will also contain a QR code superimposed (on the upper right corner) which after being decoded will provide the geographical coordinates necessary to geolocate the map.

This app (QRgeomap.com) allows to load a "QR geomap" map image file, geolocates the map and shows it on screen next to the user's live location ( ), thus facilitating a "basic navigation" with the map.

To load a map image with this app click the map button ( ) and then browse your device to select a map image.

Otherwise, if the image is hosted on the internet: qrgeomap.com can be called passing the image address (URL) as a parameter (?i=).
Ex: To load the sample map which is located at https://www.qrgeomap.com/assets/sample.png, you can use this link: https://www.qrgeomap.com/?i=https://www.qrgeomap.com/assets/sample.png

This app is built with modern web technologies that can even work offline. Add this website (qrgeomap.com) to your home screen or favorite websites of your browser so you can use it at any time to open QR geomap image files from your device.

On the other hand, this app also provides a function to create maps ( ). With this we can load a "track" file (GPX format) and generate a map with the track drawn on it (like the image above), which can be downloaded as an image for later use.

More information about the project: https://project.qrgeomap.com

Esta aplicación (QRgeomap.com) permite cargar un archivo de imagen de mapa "QR geomap", geolocaliza el mapa y lo muestra en pantalla junto a la ubicación en vivo del usuario ( ), facilitando así una "navegación básica" con el mapa.

Para cargar un mapa con esta aplicación hay que pulsar el botón de mapa ( ) y luego seleccionar una imagen de mapa que esté guardada en el dispositivo.

Otra opción, si la imagen está alojada en internet: se puede llamar a qrgeomap.com pasando por parámetro (?i=) la dirección (URL) de la imagen.
Ej: Para cargar el mapa de ejemplo que está ubicado en https://www.qrgeomap.com/assets/sample.png, se puede usar este enlace: https://www.qrgeomap.com/?i=https://www.qrgeomap.com/assets/sample.png
