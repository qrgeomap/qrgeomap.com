QR geomap is a project that aims to provide the easiest way to use and share basic maps for outdoor activities.

The idea is to use a single image file, which will logically contain the map of an area (possibly with a track, markers or any other annotations directly painted on the image), but the image will also contain a QR code superimposed (on the upper right corner) which after being decoded will provide the geographical coordinates necessary to geolocate the map.

This app (QRgeomap.com) allows to load an image file (a "QR geomap"), geolocates the map and shows it on screen next to the user's live location ( ), thus facilitating a "basic navigation" on the map.

To load a map image with this app click the map button ( ) and then browse your device to select a map image.

This app is built with modern web technologies that can even work offline, provided it has been used previously and the map image is saved on the device.

On the other hand, this app also provides a function to create maps ( ). With this we can load a "track" file (GPX format) and generate a map with the track drawn on it (like the image above), which can be downloaded as an image for later use. It also includes the option to host (publish) the generated map, obtaining a direct link so it can be easily used and shared.

Otherwise, if the image is hosted on the internet: qrgeomap.com can be called passing the image address (URL) as a parameter (?i=).
Ex: To load the sample map which is located at https://www.qrgeomap.com/assets/sample.png, you can use this link: https://www.qrgeomap.com/?i=https://www.qrgeomap.com/assets/sample.png


More information about the project: https://project.qrgeomap.com

QR geomap es un proyecto que tiene como objetivo proporcionar la forma más fácil de usar y compartir mapas básicos para actividades al aire libre.

La idea es usar un solo archivo de imagen, que lógicamente contendrá el mapa de un área (posiblemente con una ruta, marcadores o cualesquiera otras anotaciones directamente pintadas en la imagen), pero la imagen contendrá además un Código QR superpuesto (en la esquina superior derecha) que después de ser decodificado proporcionará las coordenadas geográficas necesarias para geolocalizar el mapa.

Esta aplicación (QRgeomap.com) permite cargar un archivo de imagen (un "QR geomap"), geolocaliza el mapa y lo muestra en pantalla junto a la ubicación en vivo del usuario ( ), facilitando así una "navegación básica" en el mapa.

Para cargar un mapa con esta aplicación hay que pulsar el botón de mapa ( ) y luego seleccionar una imagen de mapa que esté guardada en el dispositivo.

Esta aplicación utiliza tecnologías web modernas que permiten que pueda incluso funcionar sin conexión, siempre que se haya usado alguna vez previamente y la imagen del mapa esté guardada en el dispositivo.

Por otra parte, esta aplicación también incorpora una función para crear mapas básicos ( ). Con esto podremos cargar un archivo de "track" (formato GPX) y generar un mapa con el track dibujado sobre él (como la imagen de arriba), que se podrá descargar como imagen para su uso posterior. También incluye la opción de alojar (publicar) el mapa generado, obteniendo un enlace directo para poder utilizarlo y compartirlo fácilmente.

Otra opción, si la imagen está alojada en internet: se puede llamar a qrgeomap.com pasando por parámetro (?i=) la dirección (URL) de la imagen.
Ej: Para cargar el mapa de ejemplo que está ubicado en https://www.qrgeomap.com/assets/sample.png, se puede usar este enlace: https://www.qrgeomap.com/?i=https://www.qrgeomap.com/assets/sample.png
